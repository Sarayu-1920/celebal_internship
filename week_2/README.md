# SQL Assignment – Week 2

## Files Included

* `Sql_assignment(kaggle).ipynb` – SQL analysis performed on the Sample_Superstore dataset using SQLite.
* `Sql_assignment(word).ipynb` – Solutions for the SQL practice questions provided in the Celebal LMS assignment document.
* `Sample_Superstore.csv` – Dataset used for SQL analysis.
* `celebal_internship.db` – SQLite database created from the Sample_Superstore dataset.
* `shopease.db` – SQLite database created using the tables and sample data provided in the assignment questions.

## Tasks Done

### Sql_assignment(kaggle).ipynb

* Loaded the Sample_Superstore dataset into SQLite.
* Explored the dataset structure and records.
* Applied WHERE filters on sales, region and other columns.
* Used GROUP BY with SUM, COUNT and AVG for analysis.
* Identified top categories, products and customers.
* Performed sorting and limiting using ORDER BY and LIMIT.
* Analyzed yearly sales trends.
* Checked for duplicates and validated data quality.
* Used CASE statements for business analysis.

### Sql_assignment(word).ipynb

* Created tables with Primary Key, Foreign Key and constraints.
* Inserted sample data into all tables.
* Practiced SELECT, DISTINCT and WHERE queries.
* Applied aggregate functions such as COUNT, SUM, AVG, MIN and MAX.
* Used GROUP BY and HAVING clauses.
* Created and tested indexes for query optimization.
* Implemented INNER JOIN and LEFT JOIN queries.
* Explored Foreign Key relationships.
* Used CASE statements for data classification.
* Practiced ACID concepts and transaction handling using BEGIN, COMMIT and ROLLBACK.
