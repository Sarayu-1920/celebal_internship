# Celebal Technologies Data Engineering Internship

This repository contains my work completed during the 8-week Data Engineering Internship at Celebal Technologies.

## Internship Overview

The internship is designed to provide hands-on experience in Data Engineering concepts, SQL, Python, Big Data technologies, Spark, Databricks, and cloud-based data processing workflows.

The role focuses on building data pipelines, working with large-scale datasets, performing data transformations, ensuring data quality, and understanding modern data engineering practices.
