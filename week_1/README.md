# Week 1 Pandas Assignment

## Tasks Done

* Loaded the product dataset into a Pandas DataFrame.
* Explored the dataset using head(), tail(), shape(), columns and info().
* Checked missing values in all columns.
* Filled missing values in discount, seller_name and seller_information.
* Removed columns with a large number of missing values:

  - videos
  - variations
  - what_customers_said
* Removed duplicate records.
* Performed basic filtering and column selection.
* Cleaned the final_price column by removing currency symbols and converting it to numeric format.
* Created a new column called discount_percent using initial_price and final_price.
* Saved the cleaned dataset as cleaned_data.csv.

## Files

- assignment_1.ipynb - Notebook containing the complete solution.
- Combined_dataset.csv - Original dataset.
- cleaned_data.csv - Cleaned dataset generated after preprocessing.
